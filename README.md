# 🤖 Amazon Laptop Hunter Bot

## 😂 What is this?
Too lazy to search laptops on Amazon? Same.  
So I built a UiPath bot that does it for me… because clicking is hard.

## 🧠 What does it do?
- Goes to Amazon  
- Types “laptop” like a human (but faster 😎)  
- Stalks the first 2 laptops  
- Steals their name & price (legally… kinda)  
- Dumps everything into an Excel file like a good robot  

## ⚙️ Tech Stuff (for serious people)
- UiPath Studio  
- Data Scraping  
- Excel Activities  

## 🚀 How to run
1. Open in UiPath  
2. Hit Run  
3. Sit back and question your life choices  

## 📁 Files
- Main.xaml → The brain 🧠  
- project.json → The rules 📜  
- LaptopResults.xlsx → The treasure 💰  

## 🎯 Why did I build this?
Because manually searching is so 2005.
